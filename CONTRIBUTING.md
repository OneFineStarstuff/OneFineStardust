# Contributing to OneFineStardust

We welcome contributions to the OneFineStardust project! Whether you are fixing bugs, adding features, or improving documentation, your help is appreciated.

## How to Contribute

### 1. Fork and Clone
Fork the repository on GitHub and clone it to your local machine.

### 2. Create a Branch
Create a new branch for your changes:
```bash
git checkout -b feature/your-feature-name
```

### 3. Coding Standards
- Follow PEP 8 for Python code.
- Ensure all new features are accompanied by relevant tests or notebooks.
- Document your changes clearly in the code and update the README if necessary.

### 4. EAIP Compliance
All contributions to the core AGI framework must adhere to the **Enterprise AI Agent Interoperability Protocol (EAIP) v2.5.0**. This includes gRPC transport and SPIFFE identity requirements.

### 5. Submit a Pull Request
Once your changes are ready, submit a pull request against the `main` branch. Provide a clear description of your changes and why they are needed.

## Bug Reports
If you find a bug, please open an issue on the GitHub repository with a detailed description and steps to reproduce.

## Security
For security-related issues, please do not open a public issue. Contact the maintainers directly.
